# 04_Iterator_Practice - Master Python Iterators

## 📚 Practice Makes Perfect

This folder contains **10 practice problems + 1 bonus challenge** to master Python iterators.

| File | Difficulty | Problems |
|------|------------|----------|
| `01_beginner_problems.py` | ⭐ Beginner | Problems 1-3 |
| `02_intermediate_problems.py` | ⭐⭐ Intermediate | Problems 4-7 |

## 🎯 How to Use

1. **Try first** – Solve problems without looking at solutions
2. **Run the code** – Check if your solution works
3. **Compare** – See the provided solutions
4. **Understand** – Read the explanations

## 💡 Problem Categories

### Beginner (⭐)
- Converting iterables to iterators
- Using next() to access elements
- Manual vs automatic iteration

### Intermediate (⭐⭐)
- Multiple independent iterators
- Handling StopIteration
- Converting iterators back

### Advanced (⭐⭐⭐)
- Chaining iterators
- Custom iterator classes
- Infinite/cyclic iterators

### Real-World (🎯)
- Processing log files
- Memory-efficient large file reading
- API pagination simulation
